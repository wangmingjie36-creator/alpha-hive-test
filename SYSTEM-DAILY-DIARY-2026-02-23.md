## 📝 系统笔记

### 关键发现
1. **催化剂质量最重要** (37.75% 权重)
   - 比拥挤度更重要 (+8%)
   - 基本面 > 市场情绪

2. **拥挤度是第二因子** (29.74% 权重)
   - 避免高拥挤交易
   - 个股成功率 vs 拥挤度成反比

3. **市场情绪权重最低** (4.57% 权重)
   - 表明情绪容易变化
   - 应关注基本面驱动

### 问题与解决
1. **Google Trends API 限制**
   - 症状: 返回 HTTP 400 错误
   - 解决: 自动降级为缓存数据
   - 影响: 最小化（仅 1/6 数据源）

2. **样本不均匀分布**
   - NVDA 样本最多 (8 个)
   - 新股票样本少 (1 个)
   - 建议: 持续累积更多样本

### 优化建议
1. 增加清洁能源和新兴科技股的样本
2. 纳入期权市场数据 (隐含波动率)
